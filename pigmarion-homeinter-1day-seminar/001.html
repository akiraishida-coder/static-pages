## 全体構成：チャレンジ修了者 → オプトイン → 90分セミナー → 販売 → LINE相談

ピグマリオン恵美子／ホームインターメソッド／IQ200グローバルキッズ講座まわりの、エバーグリーン化を見据えた導線メモ。  
今回の想定は、田中側の「みんなでローンチ」型（体験セミナーで教育＋オファーまで一気に）を、ホームインター用に90分へ圧縮したセミナーを核に置く形。

### 位置づけ（既存との役割分担）

| 導線 | 役割 |
|------|------|
| 既存「3日間チャレンジプログラム」 | 新規の即集客・リスト獲得（現状どおり） |
| 本構成の主対象 | 4日間チャレンジ（Day1〜3＋追加Day4相当）を修了したハウスリスト |
| 二次オプトLP | 「もう一段深い説明会／セミナー」への再コミット（メールアドレスの再取得） |
| 5日間動画（録画・エバー） | セミナー日前後またはオプト直後に、ノウハウ系で温度を上げつつ日程に誘導（自動配信） |
| 90分セミナー | 教育パートを圧縮し、`20260402_セールススライド.md` 系の後半で診断・オファーまで |
| LINE | オプト後の同居導線。セミナー案内・リマインド・セールス後の個別相談会への誘導 |
| セールスレター | セミナー非参加層・後追い層への直接販売プッシュ（並行） |

### 推奨フロー（一文）

4日間チャレンジ修了者に対し、専用オプトインLPでメール再登録 → 同時または直後にLINE登録 → 5日間の録画動画でノウハウ＋セミナー予約促進 → 所定日程の90分ウェビナーでクロージングまで → 決済・質問はLINE個別相談会でフォロー → セミナー未視聴・未決断層にはセールスレターでリフト。

### 5日間動画の役割（エバー自動化）

- チャレンジ本編の「ライブ感」は既に体験済みのため、5日間は「復習＋抜け漏れ補完＋セミナーへの論理接続」が主。
- Day4ライブ（10分遊び・シンクフロート等）は「家庭で再現できる具体」と「ママの反応の差」を短尺で切り出しやすい。
- 各話末に「診断を取った方は○日の説明会で結果の読み解きをします」と同一メッセージで揃えると、90分セミナーへの期待が固定される。

### 教育ママ向け・紹介／集合説明会のときの注意

- 集合説明会型であれば、冒頭5分で「本日の共催・ご案内の流れ」を一言で固定し、二次オプトの理由（なぜまたメールか）をチャネル固有の一言で補足すると離脱が減る。
- 診断データの取り方・提出期限は、紹介元・コミュニティ側の運用に合わせてセミナー48時間前締切等をルール化しておく。
